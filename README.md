## Installation

Requirements:

* Python 3.13+
* [uv package manager](https://docs.astral.sh/uv/)

To install all dependencies needed to run the notebooks:

```bash
uv sync
```

## Usage

The notebooks can be run using any IDEs / code editors of your choice (PyCharm, VSCode, etc.).

If you prefer to use Jupyter Lab, you can start a local server using this command:

```bash
uv run --with jupyter jupyter lab
```
